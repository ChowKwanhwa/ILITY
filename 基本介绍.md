# ILITY 是什么？

**ILITY Network** 是一条 **L1 区块链**，定位为 **隐私保护的跨链验证与 ZK 数据互操作性框架** (Privacy-Preserving Cross-Chain Verification & ZK Data Interoperability Framework)。

### 核心价值
- **隐私验证 (Private Verification)**：允许用户在不暴露钱包地址或交易历史的情况下，证明其在多个区块链上的资产所有权和链上活动。
- **数据主权 (Data Sovereignty)**：用户可以完全自主决定其链上数据如何被证明、共享和变现。
- **统一身份 (Unified Identity)**：通过 **自主主权身份 (SSI)**，将用户分散在不同链上的多个钱包地址私密地关联在一起，形成统一的可验证身份。

### 技术工作原理
- **零知识证明 (ZKPs)**：实现“验证”与“暴露”的分离。用户可以证明自己持有某种资产或满足某种条件，而无需公开原始数据。
- **跨链集成**：支持以太坊主网、Arbitrum、zkSync、Optimism 等多种 L2 链。它将这些链的状态根和证明“桥接”到 ILITY 网络中进行标准化处理。
- **技术栈**：在 EVM 之上构建了 ZK 验证层，支持 **Groth16、PLONK 和 Halo2** 等证明方案，并利用 **EIP-4844 (Blobs)** 提高数据可用性效率。

### 生态架构组成
1. **源链 (Source Chains)**：原始数据所在地（如以太坊及各类 L2）。
2. **桥接层 (Bridging Layer)**：负责跨链消息传递（如通过 LayerZero 或 OmniBridge）。
3. **ZK 验证层 (ZK Verification Layer)**：EVM 上的智能合约，利用 BN128 和 KZG 预编译进行证明验证。
4. **身份层 (SSI Identity Layer)**：结合 **ENS** 和 **IPFS** 管理加密的身份元数据。
5. **ILITY Hub (社交交易中心)**：基于 ZK 技术的社交与信号发现平台，用户可在此进行交易并积累积分。

### ILITY Hub 与社交功能
- **Signal Discovery (信号发现)**：一个社交 Feed 流，用户可以发布基于链上数据验证的“带单信号”(CALL/PUT)，确保交易信号的真实性 (Authentic Signals)。
- **TVF (Total Value of Followers)**：衡量社交影响力的指标，结合了粉丝的总资产价值，体现了社交与金融的融合。
- **隐私资产证明**：利用 ZKP，用户可以证明自己拥有特定的资产或达到了某种交易门槛（用于获取验证蓝标或积分），但完全不需要公开自己的钱包地址。

### Airdrop Campaign (空投激励)
目前 ILITY 正在进行测试网激励活动，主要通过 **Hub**、**Discord** 和 **Zealy** 三个轨道进行：
- **Profile 积分**：通过设置个人资料并完成跨链资产验证获得。
- **Activity 积分**：通过发布高质量社交内容、使用 Trading 交互产品以及提供社区反馈获得。
- **Referral 积分**：通过邀请新用户加入平台获得。

### 应用场景
- **隐私社交与带单**：在保护个人账户隐私的前提下，证明自己的交易能力并建立社交影响力。
- **隐私 DeFi**：在不暴露余额的情况下获得借贷额度或参与激励活动。
- **合规与分析**：在保护隐私的前提下完成身份或资产验证。
